# Business-Sales-Performance-Analytics
Data Science and Analytics Future Interns internship Task 1 using power BI
## 📌 Overview
This project analyzes retail sales data to uncover trends, top-performing products, and regional performance.

## 🎯 Objectives
- Identify top revenue-generating products
- Analyze sales trends over time
- Evaluate category-wise profitability
- Understand regional sales distribution

## 🛠️ Tools Used
- Power BI
- Microsoft Excel

## 📈 Dashboard Features
- KPI Cards (Sales, Profit, Orders, AOV)
- Sales Trend Analysis
- Top 10 Products
- Category Performance (Sales vs Profit)
- Revenue Distribution by Region
- Interactive Filters (Region, Category, Date)

## 🔍 Key Insights
- Sales show an overall upward trend with seasonal peaks
- A few products contribute significantly to total revenue
- Technology category is the most profitable
- Some regions outperform others in revenue contribution
